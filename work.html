Meet App

Build a serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events. The Meet app is designed to curate events based on city inputs, with data visualization illuminating event distribution by location and popularity of event genres.
